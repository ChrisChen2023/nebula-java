Detailed Description

Base class for full and brief JSON protocols.

Simple JSON protocol has subtle differences and doesn't use it.

static final byte com.facebook.thrift.protocol.TJSONProtocolBase.getTypeIDForPeekedByte ( byte  peekedByte) throws TException
staticprotected

Given a byte, peeked from the beginning of some JSON value, determine a type of that value.

Result type is used to decide how to skip that value, so differences between compatible types don't matter and the more general type is assumed. STOP is returned for } and ] to indicate end of collection.
