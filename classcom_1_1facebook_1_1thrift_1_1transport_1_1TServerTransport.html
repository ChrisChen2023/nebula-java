void com.facebook.thrift.transport.TServerTransport.interrupt ( )

Optional method implementation.

This signals to the server transport that it should break out of any accept() or listen() that it is currently blocked on. This method, if implemented, MUST be thread safe, as it may be called from a different thread context than the other TServerTransport methods.
