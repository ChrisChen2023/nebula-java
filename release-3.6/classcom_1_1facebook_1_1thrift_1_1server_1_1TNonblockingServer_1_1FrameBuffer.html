Detailed Description

Class that implements a sort of state machine around the interaction with a client and an invoker.

It manages reading the frame size and frame data, getting it handed off as wrapped transports, and then the writing of response data back to the client. In the process it manages flipping the read and write bits on the selection key for its client.
