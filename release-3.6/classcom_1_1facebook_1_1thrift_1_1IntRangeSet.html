Detailed Description

IntRangeSet is a specialized Set<Integer> implementation designed specifically to make the generated validate() method calls faster.

It groups the set values into ranges, and in the contains() call, it does num ranges * 2 comparisons max. For the common case, which is a single, contiguous range, this approach is about 60% faster than using a HashSet. If you had a very ragged value set, like all the odd numbers, for instance, then you would end up with pretty poor running time.
