Detailed Description

Compact JSON protocol.

It is similar to simple JSON protocol, only it is JSON-compliant, and supports reading back. Unlike simple JSON protocol, compact JSON protocol encodes binary fields with Base64 encloses Infinity and -Infinity values in quotes encloses non-string map keys in quotes
